The Structure

Just folders and a SKILL.md file

mkdir -p my-plugin/.claude/skills/my-skill
cd my-plugin
touch .claude/skills/my-skill/SKILL.md
my-plugin/
└── .claude/
    └── skills/
        └── my-skill/
            └── SKILL.md

That's it. Claude finds skills in .claude/skills/ automatically.
